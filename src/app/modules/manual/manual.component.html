User Manual
1. Basic Information eXchangily, a multi-functional 3-in-1 cryptocurrency platform, provides access to a decentralized wallet, a crypto exchange, and flash payments. eXchangily’s mission is to provide its users with a full range of integrated crypto-digital currency applications and management tools to a wide variety needs including digital asset storage, management, crypto trading, investment, financial services, and other real-life applications. The platform’s key feature is decentralization. Assets will always be under the control of its users without interference from other institutions or individuals. As a result, you will not lose control of your assets, whether stored in the wallet or an exchange transaction provided that you do not share the private key. In any eXchangily wallet, the private key, a mnemonic phrase formed by a set of twelve words, is the only access point to your wallet. As long as your keys remain private, your assets will always be safe and secure.

1.1 Key Features of eXchangily

The first and only near real-time decentralized crypto exchange, eXchangily is distinguished by the following features:
  • All-in-one multi-functional platform:combination of a decentralized wallet, crypto exchange, and flash payment functions, keeping digital assets safe and transactions or flash payments convenient.
  • Powerful cross-chain functionality: cross-chain docking with multiple mainstream blockchains, the platform supports crypto storage and transactions for cryptocurrencies such as Bitcoin (BTC), Ethereum (ETH), Ethereum ERC-20 tokens, Fabcoin (FAB), Fabcoin FRC-20 tokens, Litecoin (LTC), Bitcoin Cash (BCH), Wavefield (TRX), Wavefield TRC-20 tokens, Dogecoin (DOGE), etc.
  • Easy to use: user-friendly interface similar to any centralized exchange.
  • Fast: near real-time transaction processing capabilities, allowing for a user experience similar to that of a centralized exchange.

1.2 Functions

eXchangily consists of six functional modules as follows:

  • eXchangily wallet
  • crypto exchange
  • flash payment - BindPay
  • OTC
  • blockchain browser
  • auxiliary functions

Users can navigate these sections in the menu.

The menu is divided into two areas, left and right. The left side comprises the decentralized core functions and the right side houses the centralized auxiliary functions.

The sections in the key functional areas are Wallet, Market, Trade, Explore, OTC Trade, Mint Token, and BindPay. The wallet functions as an independent decentralized wallet, while market, trade, and flash payments (BindPay) all are decentralized exchange functions. The blockchain browser shows the records of all transactions on the blockchain, and fiat currency trading is a semi-decentralized function that provides transactions between fiat currency and cryptocurrency.

Auxiliary functions include an introduction about the project, news updates, service support, community communications, events, and settings. These help users navigate eXchangily and communicate important updates and events.


2. Exchangily Wallet Wallets are the gateway to cryptocurrency; the first step to get involved is creating and owning a wallet.

2.1 Create a wallet

To create a wallet on the eXchangily platform, select Wallet in the menu navigation bar. If you are visiting the website for the first time on your device, a randomly generated set of words called a mnemonic phrase will pop up automatically. Copy these words precisely and make sure to keep them in a safe location. The best practice is to physically write it out and store it safely. In the case where a key is kept electronically and something happens to the device, you may lose the mnemonic and thus, access to the wallet. (The image below shows an example of a mnemonic phrase.)

Once copied, press Next to enter and verify the key.

After entering the mnemonic, Confirm. If you click Back on this page, a new set of words will generate and you will have to recopy them.

Next, name the wallet and set a password. The wallet names are for reference purposes only, and in the case that you have multiple wallets, you can use the names to distinguish between the private keys. (There are many reasons for multiple wallets such as a cold wallet, where you only deposit money and do not withdraw, or a hot wallet, which serves as a checking account and sees many transactions.)

When creating the main wallet password, choose a strong password that fits the following requirements: eight or more characters, including at least one uppercase letter, one lowercase letter, one number, and one special symbol (! @#$%^&*). The password functions only as encryption on your device for the mnemonics to prevent hackers from accessing the wallet. Since the password is specific to the device it was created on and not stored externally on a remote server, you cannot recover or access the wallet via the wallet name and password on other devices.

The wallet name and password do not need to be memorized since you can recover a wallet with your mnemonic private key and reset the wallet name and password.

Click on Submit to complete!

2.2 Restoring a Wallet

You can restore your wallet if it was deleted or is on another device. There are two methods, dependent on if your device has an existing wallet:

  • If your device does not have a wallet recorded, the Create New Wallet page will pop up automatically. Select Existing Mnemonics? Restore, then hit Restore to reach the wallet recovery page.
  • If another wallet exists on the device, select Import Wallet in the My Wallet function bar on the right side of the main wallet page.

Rename the wallet and reset the password – they can be the same or different from the previous name and password. Type the mnemonics with each word separated by a space. Once the set of words is in, select Restore Wallet to complete the process.

2.3.2 Common wallet operation Guidelines

Common wallet functions include money transactions, adding gas, coin deposits, transaction history, adding a display password to the wallet, wallet deletion, smart contract operations, etc.

2.3.2.1 Receiving money

Select Receive in the summary area to open the Receive window. Once you choose the currency that you will receive, the corresponding wallet address and QR code will show. To send the wallet address to the other party, you can copy and send it to them or they can also scan the QR code directly. The payment address is not confidential, similar to a bank account number, so you can send it via text message, email, WeChat, or any social platform.

2.3.2.2 Payment

Select Send in the summary area and the payment window will pop up. To send currency, make sure that the recipient’s wallet address is accurate. Then, choose the currency, specify the coin quantity, write a quick note on the transaction, and select Send.

Few things about payments:
After selecting the currency, make sure that there are sufficient chain coins in the wallet to pay the transaction fee. (e.g. If you want to transfer USDT (ERC-20) coins, allocate enough ETH to pay the Ether blockchain transaction fee. If you want to transfer USDT (TRC-20), ensure there is enough TRX to pay the wave field transaction fee. Similarly, transferring EXG, DUSD, BST, and DSC will require enough FAB to pay the transaction fee.)
Usually, the default transaction fee of the system is sufficient to process the payment. However, sometimes blockchain transactions are frequent and volatile, and the default transaction fee may be too low. In those cases, transactions may not be completed instantly (common on Ethereum or Bitcoin blockchain). To prevent this, you can set a higher transaction fee rate by toggling Advanced.

Enter the wallet password and select Confirm to complete the transfer. Note: After the transfer is complete, the processing time is dependent on the corresponding blockchain. Some blockchains are fast and the transaction will complete in seconds, but others are slower and can take a few minutes, hours, or even days. You can click on the transaction number in Transaction History for status updates.

2.3.2.3 Add Gas to the Exchange

To use the exchange, you will need FAB first since all actions will require gas (e.g. depositing coins). Since eXchangily uses FAB coins as its gas, you will need to add FAB coins to your wallet.

How do I get FAB coins? There is a multitude of ways to add gas. eXchangily sometimes offers free FAB, but you can also buy gas from others, buy via fiat coins through the OTC channel, or mine them yourself. To use the exchange, you will need FAB coins.

If you already have FAB in your wallet, select Add Gas to bring up the Add Gas window. Fill in the amount of Gas you want to add. A few FAB coins are usually sufficient to handle multiple transactions. Select Confirm, and verify with your password. You may have to wait a few minutes for the gas balance number to update before you can deposit coins to the exchange.

2.3.2.4 Deposit coins to eXchangily

The trading area is separate from the wallet, although still controlled by the same mnemonic. To trade a particular coin, you must first move that coin from your wallet to the exchange. Note: eXchangily is a decentralized exchange and coins transferred to the exchange will still be controlled by your wallet's private key. No one can touch your assets, whether they are pending or canceled since they are controlled by your private key.

In the asset list area, the down arrow at the end of each coin line will confirm if it can deposit into the exchange. If no down arrow shows, there is not sufficient gas to complete the transaction and you will need to add gas.

Transferring coins to an exchange is a cross-chain operation. In addition to eXchangily having enough FAB, the wallet must also have sufficient coins of the transferring blockchain to pay that blockchain’s fees. (e.g. If you want to transfer USDT (ERC-20) coins, allocate enough ETH to pay the Ether blockchain transaction fee. If you want to transfer USDT (TRC-20), ensure there is enough TRX to pay the wave field transaction fee. Similarly, transferring EXG, DUSD, BST, and DSC will require enough FAB to pay the transaction fee.)

The down arrow will open the deposit window. Enter the quantity to be deposited, select Confirm, and verify by reentering your password.

NOTE: Because depositing coins to the exchange is a cross-chain operation, you will need to wait for both blockchains to process before coins will show in exchange and be available to trade.

2.3.2.5 Display settings

The system can also set a display password aside from the main password set at wallet creation. The display password hides and shows the wallet information and display settings. The main password encrypts the mnemonics and is required to change both the main and display passwords. If no display password exists, it will default to the main password.
